# Roadmap: Coffee-ERP-Ops (v3.7)

## 1. Стратегическое видение

Создание «Операционного монитора» для кофейного бизнеса, который автоматизирует рутину без претензии на бухгалтерский учет.

## 2. Обновленный Технологический Стек

- **Frontend:** React + SCSS Modules + Radix UI (Headless) + clsx.
- **Backend:** Node.js + Prisma + TRPC.
- **Инфраструктура:** Local Storage (/public/uploads), GPS & EXIF Validation.

## 3. Ключевые этапы (Milestones)

### Квартал 1: Фундамент и Персонал

- [ ] Запуск Core-логики (Бизнесы, Юниты) и Ownership Handshake.
- [ ] Модуль Графиков с защитой Double Shift Guard и IP-Geofencing.

### Квартал 2: Производство и Склад

- [ ] Stateless Inventory и Snapshot Engine (Stateless логика).
- [ ] Smart TTK с каскадным пересчетом COGS и картой аллергенов.

### Квартал 3: Контроль и Качество

- [ ] Операционный контроль (Чек-листы) с True Capture и GPS-метками.
- [ ] LMS с автоматической генерацией тестов по ТТК.

### Квартал 4: Аналитика и Активы

- [ ] Управление оборудованием с блокировкой продаж при авариях (Safety Lock).
- [ ] Управленческий учет (Cash Management) с механизмом Сторнирования.
