# Style Guide: Системная архитектура и визуальный язык

Настоящий документ регламентирует стандарты стилизации Coffee-ERP-Ops.
Все визуальные решения должны базироваться на системных токенах из `_variables.scss` и системных миксинах из `_mixins.scss`.

---

## 1. Цветовая палитра (Zero-Hex Policy)

| Переменная       | Значение (Hex/RGBA)        | Описание (Tailwind Equiv.)            |
| :--------------- | :------------------------- | :------------------------------------ |
| `$bgCanvas`      | `#020617`                  | Основной фон (Slate-950).             |
| `$bgSurface`     | `rgba(15, 23, 42, 0.6)`    | База для Glass-элементов (Slate-900). |
| `$accentPrimary` | `#10b981`                  | Акцент (Emerald-500). Успех, фокус.   |
| `$accentWarning` | `#f59e0b`                  | Предупреждения (Amber-500).           |
| `$accentDanger`  | `#f43f5e`                  | Ошибки/HACCP (Rose-500).              |
| `$textPrimary`   | `#f8fafc`                  | Текст Slate-50 (Контраст 18.9:1).     |
| `$textDimmed`    | `#94a3b8`                  | Вторичный текст (Slate-400).          |
| `$borderGlass`   | `rgba(255, 255, 255, 0.1)` | Границы Glass-карточек.               |

Цвета приведены для примера. все цвета должны быть rgba

### 1.1. Спецэффекты (Glassmorphism)

Система базируется на следующих константах:

- **Blur**: `$blurMd: 12px` (стандарт), `$blurLg: 20px` (оверлеи).
- **Radius**: `$radiusMain: 16px` (единый стандарт скругления).
- **Z-Index**:
  - `$zBelow: -1` (Ambient Glow орбы).
  - `$zBase: 0` (Контент).
  - `$zAbove: 10` (Floating UI).
  - `$zOverlay: 50` (Modals/Drawers).

## 2. Методология Sass

### 2.1. Использование @use

Мы полностью отказались от `@import` в пользу модуля `@use`.

- Глобальные переменные внедряются через `vite.config.ts`.
- Внутри самих файлов определений (например, `_mixins.scss`) необходимо явно
  подключать переменные: `@use "variables" as *;`.

### 2.2. Системные миксины

| Миксин       | Параметры           | Назначение                                        |
| :----------- | :------------------ | :------------------------------------------------ |
| `glassCard`  | `($opacity, $blur)` | Эффект матового стекла с обводкой.                |
| `radialGlow` | `($color)`          | Свечение, следующее за курсором (`--mouseX`).     |
| `fluidText`  | `($min, $max)`      | Адаптивный шрифт через `clamp()`.                 |
| `dataText`   | `($size)`           | Моноширинный текст (ID, цены) на базе Geist Mono. |

---

## 3. Полировка (Micro-interactions)

- **Transitions**: Использовать только `$transitionMain` (`cubic-bezier(0.4, 0, 0.2, 1)`).
- **Ambient Glow**: Световые орбы в `main.scss` должны иметь `opacity: 0.12` и `filter: blur(120px)`.
- **Selection**: Цвет выделения текста переопределен глобально под стиль Dark Emerald.
